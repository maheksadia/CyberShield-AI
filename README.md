# 🛡 CyberShield AI

AI-powered cybersecurity dashboard built with React and Vite.

CyberShield AI helps users analyze password strength, detect suspicious URLs, and monitor security threats through an interactive dashboard.

## 🚀 Features

- 🔐 Password Strength Scanner
- 🌐 Suspicious URL Detection
- 🚨 Threat Monitoring System
- 📊 Security Score Analysis
- ⚡ Fast React + Vite Interface

## 🛠 Tech Stack

- React.js
- Vite
- JavaScript
- CSS
